Scenario vs Budgeted Profits – Sensitivity Analysis

This sample shows how to determine the effect on Total Portfolio Scenario vs Budgeted Profits, from:

(i) Discounting the price of a selection of Products,
(ii) Reducing the costs of all products, and/or
(iii) Changing the demand of all products

Data Model - Snippet

Measures - Actual

Budget Profit (1 Yr r/f)

DAX 
Budget Profit (1 Yr r/f) = 
[Budget Sales (1 Yr r/f)] - [Budgeted Costs (1 Yr r/f)]

Budget Sales (1 Yr r/f)

DAX 
Budget Sales (1 Yr r/f) =
CALCULATE (
    CALCULATE ( [Total Sales], DATEADD ( Calendar_Table[Date], -1, YEAR ) ),
    ALL ( Products_Table[Products] )
)

Budgeted Costs (1 Yr r/f)

DAX 
Budgeted Costs (1 Yr r/f) = 
CALCULATE (
    CALCULATE ( [Total Costs], DATEADD ( Calendar_Table[Date], -1, YEAR ) ),
    ALL ( Products_Table[Products] )
)

Measures - Comparison

[Profit Difference [%]]: Scenarios vs Budgeted]

DAX 
[Profit Difference [%]]: Scenarios vs Budgeted] = 
DIVIDE([Scenario Profits (1 Yr r/f)] - [Budget Profit (1 Yr r/f)], [Budget Profit (1 Yr r/f)], 0)

Measures - Scenario

Discounted Sales

DAX 
Discounted Sales = 
VAR DiscountedProducts =
    INTERSECT (
        VALUES ( Products_Table[Products] ),
        ALL ( Products_Table[Products] )
    )
RETURN
    CALCULATE (
        SUMX (
            Sales_Table,
            Sales_Table[Quantity] * ( 1 + [Demand Changes Value] ) * Sales_Table[Unit Price] * ( 1 + [Discounting Value] )
        ),
        DiscountedProducts
    )

Discounted Sales (1 Yr r/f)

DAX 
Discounted Sales (1 Yr r/f) = 
CALCULATE ([Discounted Sales], DATEADD(Calendar_Table[Date], -1, YEAR))

Non Discounted Sales

DAX 
Non Discounted Sales = 
VAR NonDiscountedProducts =
    EXCEPT ( ALL ( Products_Table[Products] ), VALUES ( Products_Table[Products] ) )
RETURN
    CALCULATE (
        SUMX (
            Sales_Table,
            Sales_Table[Quantity] * ( 1 + [Demand Changes Value] ) * Sales_Table[Unit Price]
        ),
        NonDiscountedProducts
    )

Non Discounted Sales (1 Yr r/f)

DAX 
Non Discounted Sales (1 Yr r/f) = 
CALCULATE ([Non Discounted Sales], DATEADD(Calendar_Table[Date], -1, YEAR))

Scenario Costs

DAX 
Scenario Costs =
CALCULATE (
    SUMX (
        Sales_Table,
        Sales_Table[Quantity] * ( 1 + [Demand Changes Value] ) * Sales_Table[Unit Cost]
    ) * ( 1 + [Cost Price Change Value] ),
    ALL ( Products_Table[Products] )
)

Scenario Costs (1 Yr r/f)

DAX 
Scenario Costs (1 Yr r/f) = 
CALCULATE ([Scenario Costs], DATEADD(Calendar_Table[Date], -1, YEAR))

Scenario Profit Margin

DAX 
Scenario Profit Margin = 
DIVIDE ([Scenario Profits (1 Yr r/f)], [Scenario Sales (1 Yr r/f)], 0)

Scenario Profits (1 Yr r/f)

DAX 
Scenario Profits (1 Yr r/f) =
[Scenario Sales (1 Yr r/f)] - [Scenario Costs (1 Yr r/f)]

Scenario Sales (1 Yr r/f)

DAX 
Scenario Sales (1 Yr r/f) =
[Discounted Sales (1 Yr r/f)] + [Non Discounted Sales (1 Yr r/f)]

Total Costs

DAX 
Total Costs = 
SUMX ( Sales_Table, Sales_Table[Unit Cost] * Sales_Table[Quantity] )

Total Sales

DAX 
Total Sales = 
SUMX ( Sales_Table, Sales_Table[Unit Price] * Sales_Table[Quantity] )

Tables

Total Sales

DAX 
Calendar_Table =
GENERATE (
    CALENDAR ( DATE ( 2017, 1, 1 ), DATE ( 2019, 12, 31 ) ),
    VAR currentDay = [Date]
    VAR Day =
        DAY ( currentDay )
    VAR Month =
        MONTH ( currentDay )
    VAR Year =
        YEAR ( currentDay )
    RETURN
        ROW ( "Day", day, "Month", month, "Year", year )
)

Cost Price Change

DAX 
Cost Price Change =
GENERATESERIES ( -0.1, 0.2, 0.05 )

Demand Changes

DAX 
Demand Changes =
GENERATESERIES ( -0.06, 0.2, 0.02 )

Discounting

DAX 
Discounting =
GENERATESERIES ( -0.3, 0.02, 0.02 )
