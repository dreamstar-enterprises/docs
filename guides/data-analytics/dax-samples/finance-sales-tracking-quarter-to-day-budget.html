Finance - Sales Tracking â€“ Quarterly to Daily Budget

In this sample, we track Actual Sales against Budgeted Sales. This is relatively easy to do. The only slight complication is that the Budget has been set Quarterly, yet we want to also track differences at a day, week, or monthly level. To do this, we need to allocate each Quarterâ€™s Budget to each month, week, and day, taking into account the number of working days, in each selected time period.

Data Inputs

For this sample, we take the following data inputs:

Budget

As stated earlier, this has been set each quarter, rather than each week, or month

Actuals Data

This is set to us in the following format â€“ with sales recorded on whichever day that happen to relate to.

Measures

IsBudgetValid

DAX 
IsBudgetValid :=
NOT (
    ISFILTERED ( 'Date'[Date] ) || ISFILTERED ( 'Date'[Month] )
        || ISFILTERED ( 'Date'[Week] )
        || ISFILTERED ( 'Date'[Day] )
)

Budget

DAX 
Budget :=
IF ( [IsBudgetValid], [BudgetCalc], [AllocatedBudget] )

BudgetCalc

DAX 
BudgetCalc :=
CALCULATE (
    SUM ( Quarterly_Budget[BudgetAmount] ),
    FILTER (
        ALL ( Quarterly_Budget[YearQuarter] ),
        CONTAINS (
            VALUES ( 'Date'[YearQuarter] ),
            'Date'[YearQuarter], Quarterly_Budget[YearQuarter]
        )
    ),
    FILTER (
        ALL ( Quarterly_Budget[Item] ),
        CONTAINS ( VALUES ( Items[Item] ), Items[Item], Quarterly_Budget[Item] )
    )
)

AllocatedBudget

DAX 
AllocatedBudget :=
SUMX (
    KEEPFILTERS (
        CROSSJOIN ( VALUES ( 'Date'[YearQuarter] ), VALUES ( Items[Item] ) )
    ),
    [AllocationFactor] * [BudgetCalc]
)

AllocationFactor

DAX 
AllocationFactor :=
[WorkingDays]
    / CALCULATE (
        [WorkingDays],
        ALLEXCEPT ( Items, Items[Item] ),
        ALL ( 'Date' ),
        VALUES ( 'Date'[YearQuarter] )
    )

WorkingDays

DAX 
WorkingDays :=
SUMX (
    VALUES ( 'Date'[Year] ),
    CALCULATE ( COUNTROWS ( 'Date' ), 'Date'[WorkingDay] = TRUE )
)

Actuals

DAX 
Actuals :=
SUM ( Actuals[ActualAmount] )

Var%

DAX 
Var% :=
DIVIDE ( [Actuals] - [Budget], [Budget] )

YTDBudget

DAX 
YTDBudget :=
TOTALYTD ( [Budget], 'Date'[Date] )

YTDActuals

DAX 
YTDActuals :=
TOTALYTD ( [Actuals], 'Date'[Date] )

YTDVar%

DAX 
YTDVar% :=
DIVIDE ( [YTDActuals] - [YTDBudget], [YTDBudget] )

QTDBudget

DAX 
QTDBudget :=
IF ( ISFILTERED ( [Date] ), TOTALQTD ( [Budget], 'Date'[Date] ), [Budget] )

QTDActuals

DAX 
QTDActuals :=
IF ( ISFILTERED ( [Date] ), TOTALQTD ( [Actuals], 'Date'[Date] ), [Actuals] )

QTDVar%

DAX 
QTDVar% :=
DIVIDE ( [QTDActuals] - [QTDBudget], [QTDBudget] )

Calculated Columns

YearQuarter

DAX 
YearQuarter =
Quarterly_Budget[Year] & Quarterly_Budget[Quarter]

Year

DAX 
Year =
YEAR ( 'Date'[Date] )

Quarter

DAX 
Quarter =
"Q" & FORMAT ( 'Date'[Date], "Q" )

YearQuarter

DAX 
YearQuarter =
'Date'[Year] & 'Date'[Quarter]

MonthNumber

DAX 
MonthNumber =
MONTH ( 'Date'[Date] )

Month

DAX 
Month =
FORMAT ( 'Date'[Date], "mmmm" )

Week

DAX 
Week =
WEEKNUM ( 'Date'[Date] )

Day

DAX 
Day =
DAY ( 'Date'[Date] )

WorkingDay

DAX 
WorkingDay =
WEEKDAY ( [Date], 2 ) < 6
