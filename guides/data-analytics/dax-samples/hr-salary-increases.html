HR - Salary Increases

This sample provides insights from the following Salary & Salary Increase datasets:

Salary & Salary Increase (data)

Click to view large-sized image

Click to view large-sized image

Salary Increase Measures

The following measures are shown:

  1. HeadCount by Gender
  2. Total EoM FTE Salary by Gender
  3. Average EoM FTE Salary per Person by Gender
  4. Total SoM FTE Salary
  5. Total FTE Salary Change
  6. Count of Staff with FTE Salary Change by Gender, by Reason
  7. Average FTE Salary Change per Person by Gender, by Reason
  8. Total FTE Salary Change by Gender, by Month
  9. Average FTE Salary Change per Person by Gender, by Month
  10. Total FTE Salary Change by Reason, by Month
  11. Mean / IQR EoM FTE Salary per Person by Gender
  12. Total FTE Salary Change by Eligibility, by Month

Model Extensions

Possible extensions to this model, could include:

  • Inclusion of ‘Freelancers’ to all datasets, to help distinguish from ‘Permanent’ staff. This could be done by adding an extra field called ‘Freelancers vs Permanent’.
  • Linking Freelance records automatically to their Limited Company or Agency Timesheet submissions.
  • Adjusting of calculations accordingly.

Measures

Average FTE Salary Change by Person GBP

DAX 
Average FTE Salary Change by Person GBP = 
CALCULATE (
    AVERAGEX (
        SUMMARIZE (
            'Cross Joined Staff Table (DAX)',
            [User ID],
            [Start of Month],
            "Average Salary Change",
                CALCULATE (
                    AVERAGE ( 'Cross Joined Staff Table (DAX)'[Running Salary Change in Month GBP] ),
                    FILTER (
                        'Cross Joined Staff Table (DAX)',
                        'Cross Joined Staff Table (DAX)'[Running Salary Change in Month GBP] <> 0
                            && 'Cross Joined Staff Table (DAX)'[Part of HeadCount at End of Month] = "Y"
                    )
                )
        ),
        [Average Salary Change]
    ),
    FILTER (
        'Cross Joined Staff Table (DAX)',
        'Cross Joined Staff Table (DAX)'[Start of Month] <= TODAY ()
    )
)

Average FTE Salary Change by Reason by Person GBP

DAX 
Average FTE Salary Change by Reason by Person GBP = 
CALCULATE (
    AVERAGEX (
        SUMMARIZE (
            'Cross Joined Staff Table (DAX)',
            'Cross Joined Staff Table (DAX)'[User ID],
            'Cross Joined Staff Table (DAX)'[Reason for Salary Increase],
            'Cross Joined Staff Table (DAX)'[Start of Month],
            "Average Salary Change",
                CALCULATE (
                    AVERAGE ( 'Cross Joined Staff Table (DAX)'[Running Salary Change by Reason in Month GBP] ),
                    FILTER (
                        'Cross Joined Staff Table (DAX)',
                        'Cross Joined Staff Table (DAX)'[Start Date]
                            = MAX ( 'Cross Joined Staff Table (DAX)'[Start Date] )
                    )
                )
        ),
        [Average Salary Change]
    ),
    FILTER (
        'Cross Joined Staff Table (DAX)',
        'Cross Joined Staff Table (DAX)'[Start of Month] <= TODAY ()
            && 'Cross Joined Staff Table (DAX)'[Running Salary Change by Reason in Month GBP]
                <> BLANK ()
    )
)

Tables

Calendar

DAX 
Calendar = 
VAR MinDate =
    MIN ( Min_Date[Min_Date] )
VAR MaxDate =
    MAX ( Max_Date[Max_Date] )
RETURN
    CALENDAR ( EOMONTH ( MinDate, - 1 ) + 1, EOMONTH ( MaxDate, 0 ) )

Cross Joined Staff Table (DAX)

DAX 
Cross Joined Staff Table (DAX) = 
CROSSJOIN('Monthly Calendar', Inventory_Table)

Calculated Columns

Weekend

DAX 
Weekend = 
SWITCH ( WEEKDAY ( [Date] ), 1, 1, 7, 1, 0 )

Days since previous increase

DAX 
Days since previous increase = 
VAR StartDateofPreviousSalaryIncrease =
    IF (
        'Cross Joined Staff Table (DAX)'[Start Date] >= 'Cross Joined Staff Table (DAX)'[Start of Month],
        CALCULATE (
            MAX ( [Start Date] ),
            FILTER (
                ALLEXCEPT (
                    'Cross Joined Staff Table (DAX)',
                    'Cross Joined Staff Table (DAX)'[User ID],
                    'Cross Joined Staff Table (DAX)'[Start of Month]
                ),
                'Cross Joined Staff Table (DAX)'[Salary Increase ?] = "Salary Increase"
                    && 'Cross Joined Staff Table (DAX)'[Start Date]
                        < EARLIER ( 'Cross Joined Staff Table (DAX)'[Start Date] )
            )
        ),
        CALCULATE (
            MAX ( [Start Date] ),
            FILTER (
                ALLEXCEPT (
                    'Cross Joined Staff Table (DAX)',
                    'Cross Joined Staff Table (DAX)'[User ID],
                    'Cross Joined Staff Table (DAX)'[Start of Month]
                ),
                'Cross Joined Staff Table (DAX)'[Salary Increase ?] = "Salary Increase"
                    && 'Cross Joined Staff Table (DAX)'[Start Date]
                        <= EARLIER ( 'Cross Joined Staff Table (DAX)'[Start Date] )
            )
        )
    )
VAR StartDateofPreviousSalaryIncreaseCheck =
    IF (
        ISBLANK ( StartDateofPreviousSalaryIncrease ),
        CALCULATE (
            MAX ( [Start Date] ),
            FILTER (
                ALLEXCEPT (
                    'Cross Joined Staff Table (DAX)',
                    'Cross Joined Staff Table (DAX)'[User ID],
                    'Cross Joined Staff Table (DAX)'[Start of Month]
                ),
                'Cross Joined Staff Table (DAX)'[FTE Salary GBP]
                    <= EARLIER ( 'Cross Joined Staff Table (DAX)'[FTE Salary GBP] )
                    && 'Cross Joined Staff Table (DAX)'[Start Date]
                        < EARLIER ( 'Cross Joined Staff Table (DAX)'[Start Date] )
            )
        ),
        StartDateofPreviousSalaryIncrease
    )
VAR PreviousSalaryIncrease =
    CALCULATE (
        MAX ( [FTE Salary GBP] ),
        FILTER (
            ALLEXCEPT (
                'Cross Joined Staff Table (DAX)',
                'Cross Joined Staff Table (DAX)'[User ID],
                'Cross Joined Staff Table (DAX)'[Start of Month]
            ),
            [Start Date] = StartDateofPreviousSalaryIncreaseCheck
        )
    )
VAR MinStartDatewithPreviousSalaryIncrease =
    CALCULATE (
        MIN ( [Start Date] ),
        FILTER (
            ALLEXCEPT (
                'Cross Joined Staff Table (DAX)',
                'Cross Joined Staff Table (DAX)'[User ID],
                'Cross Joined Staff Table (DAX)'[Start of Month]
            ),
            [FTE Salary GBP] = PreviousSalaryIncrease
        )
    )
VAR ComparisonDate =
    IF (
        AND (
            'Cross Joined Staff Table (DAX)'[Start Date] <= 'Cross Joined Staff Table (DAX)'[End of Month],
            'Cross Joined Staff Table (DAX)'[Start Date] >= 'Cross Joined Staff Table (DAX)'[Start of Month]
        ),
        'Cross Joined Staff Table (DAX)'[Start Date],
        MIN (
            'Cross Joined Staff Table (DAX)'[End of Month],
            'Cross Joined Staff Table (DAX)'[End Date Fill]
        )
    )
RETURN
    IF (
        OR (
            'Cross Joined Staff Table (DAX)'[Part of HeadCount at End of Month] = "Y",
            'Cross Joined Staff Table (DAX)'[Started in Month ?] = "Y"
        ),
        IF (
            OR (
                ComparisonDate < MinStartDatewithPreviousSalaryIncrease,
                ISBLANK ( MinStartDatewithPreviousSalaryIncrease )
            ),
            BLANK (),
            ComparisonDate - MinStartDatewithPreviousSalaryIncrease + 1
        ),
        BLANK ()
    )

Eligible Increase ?

DAX 
Eligible Increase ? = 
IF (
    OR (
        'Cross Joined Staff Table (DAX)'[Part of HeadCount at End of Month] = "Y",
        'Cross Joined Staff Table (DAX)'[Started in Month ?] = "Y"
    ),
    IF (
        'Cross Joined Staff Table (DAX)'[Days since previous increase] > 30,
        "Eligible",
        "Not Eligible"
    ),
    BLANK ()
)

End Date Fill

DAX 
End Date Fill = 
IF ( ISBLANK ( [End Date] ), 2958100, [End Date] )

EOM FTE Salary GBP

DAX 
EOM FTE Salary GBP = 
IF (
    'Cross Joined Staff Table (DAX)'[Part of HeadCount at End of Month] = "N",

    CALCULATE (
        MAX ( [FTE Salary GBP] ),
        FILTER (
            ALLEXCEPT (
                'Cross Joined Staff Table (DAX)',
                'Cross Joined Staff Table (DAX)'[User ID],
                'Cross Joined Staff Table (DAX)'[Start of Month]
            ),
            'Cross Joined Staff Table (DAX)'[Part of HeadCount at End of Month] = "Y"
        )
    )
)

DAX 
FTE Salary Change GBP =

Running Salary Change by Reason in Month GBP

DAX 
Running Salary Change by Reason in Month GBP = 
VAR FilteredTable =
    CALCULATETABLE (
        'Cross Joined Staff Table (DAX)',
        FILTER (
            ALLEXCEPT (
                'Cross Joined Staff Table (DAX)',
                'Cross Joined Staff Table (DAX)'[User ID],
                'Cross Joined Staff Table (DAX)'[Start of Month],
                'Cross Joined Staff Table (DAX)'[Reason for Salary Increase]
            ),
            'Cross Joined Staff Table (DAX)'[Started in Month ?] = "Y"
                && 'Cross Joined Staff Table (DAX)'[Start Date]
                    <= EARLIER ( 'Cross Joined Staff Table (DAX)'[Start Date] )
        )
    )
VAR SalaryChangeInMonth =
    SUMX ( FilteredTable, 'Cross Joined Staff Table (DAX)'[FTE Salary Change GBP] )
RETURN
    IF (
        OR (
            SalaryChangeInMonth = 0,
            'Cross Joined Staff Table (DAX)'[Started in Month ?] <> "Y"
        ),
        BLANK (),
        SalaryChangeInMonth
    )

                'Cross Joined Staff Table (DAX)'[User ID],
