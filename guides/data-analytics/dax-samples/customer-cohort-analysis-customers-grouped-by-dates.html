Customer Cohort Analysis – Grouping of Customers into Monthly Cohorts based on their purchase dates

This sample:

  • Groups each customer into a “Join Month Cohort”, based on the month in which they started / made their first purchase

  • For each “Join Month Cohort”, it then groups customers in that Cohort, into a Monthly Rentention Period Group, based on how long that customer’s Rention Period is, or was.

  • A Customer’s Retention Period is the number of days between a Customer’s Onboarding date, and their next (or Second Purchase)

Data Model - Snippet

Measures - Cohort

Cohort Customers

DAX 
Cohort Customers =
COUNTROWS ( Customer_Table )

Cohort Month

DAX 
Cohort Month =
VAR JoinDate = [Join Date]
RETURN
    CALCULATE (
        SELECTEDVALUE ( Calendar_Table[Month & Year] ),
        FILTER ( ALL ( Calendar_Table ), Calendar_Table[Date] = JoinDate )
    )
// works in customer context

Cohort Start Month

DAX 
Cohort Start Month =
VAR OnboardingDate = [Onboarding Date]
RETURN
    CALCULATE (
        SELECTEDVALUE ( Calendar_Table[Month & Year] ),
        FILTER ( ALL ( Calendar_Table ), Calendar_Table[Date] = [Onboarding Date] )
    )

Join Date

DAX 
Join Date =
MIN ( Sales_Table[Date] )
// works in Customer Context

No. of Unique Customers as a % of all Customers

DAX 
No. of Unique Customers as a % of all Customers =
CALCULATE (
    DIVIDE ( [No. of Unique Customers in Retention Period], [Unique Customers], 0 ),
    TREATAS (
        VALUES ( Calendar_Table[Month & Year] ),
        Customer_Table[Join Month Cohort]
    )
)

No. of Unique Customers in Retention Period

DAX 
No. of Unique Customers in Retention Period =
VAR CustomerDimension =
    VALUES ( Customer_Table[Customer] )
RETURN
    IF (
        SELECTEDVALUE ( 'Retention Period Monthly Cohorts'[Period] ) = 0,
        [Unique Customers],
        CALCULATE (
            CALCULATE (
                [Unique Customers],
                FILTER (
                    CustomerDimension,
                    COUNTROWS (
                        FILTER (
                            'Retention Period Monthly Cohorts',
                            [Retention Days] > 'Retention Period Monthly Cohorts'[Min Days]
                                && [Retention Days] <= 'Retention Period Monthly Cohorts'[Max Days]
                        )
                    ) > 0
                )
            ),
            TREATAS (
                VALUES ( Calendar_Table[Month & Year] ),
                Customer_Table[Join Month Cohort]
            )
        )
    )

Onboarding Date

DAX 
Onboarding Date =
CALCULATE (
    [Join Date],
    ALL ( Sales_Table ),
    VALUES ( Customer_Table[Customer] )
)

Retention Days

DAX 
Retention Days =
VALUE ( [Second Purchase Date] ) - VALUE ( [Onboarding Date] )

Second Purchase Date

DAX 
Second Purchase Date =
CALCULATE (
    MIN ( Sales_Table[Date] ),
    FILTER ( ALL ( Sales_Table ), Sales_Table[Date] > [Onboarding Date] ),
    VALUES ( Customer_Table[Customer] )
)
//looking for the min sale AFTER the initial first sale

Unique Customers

DAX 
Unique Customers =
COUNTROWS ( VALUES ( Sales_Table[Customer] ) )

Tables

Calendar_Table

DAX 
Calendar_Table =
GENERATE (
    CALENDAR ( DATE ( 2017, 1, 1 ), DATE ( 2019, 12, 31 ) ),
    VAR currentDay = [Date]
    VAR Day =
        DAY ( currentDay )
    VAR Month =
        MONTH ( currentDay )
    VAR Year =
        YEAR ( currentDay )
    RETURN
        ROW ( "Day", day, "Month", month, "Year", year )
)

Calculated Columns

Join Date

DAX 
Join Date =
[Join Date]

Join Month Cohort

DAX 
Join Month Cohort =
[Cohort Month]

Join Month Index

DAX 
Join Month Index =
CALCULATE (
    SELECTEDVALUE ( Calendar_Table[YearMonthInt] ) * 100,
    FILTER (
        Calendar_Table,
        Calendar_Table[Month & Year] = 'Cohort Measures'[Cohort Start Month]
    )
)
